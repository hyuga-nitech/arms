# arms/Python
<img src="https://img.shields.io/badge/Python-3.10.4-blue?&logo=Python">

xArmとmikataArmを接続し、多自由度化を目指すプロジェクトのリポジトリです．
複数人の操作者の手の動きを２つのアームに分散して反映させ、操作を行います．
