# arms/Python
<img src="https://img.shields.io/badge/Python-3.10.4-blue?&logo=Python">

mikataArmnのみで動作の安定化を図るプロジェクトのリポジトリです．
単一の操作者の手の動きを反映させ、操作を行います．
mikataArmの各種モードのテスト、Thread処理のテストが目的です．
余計な機能は削除してあり、汎用性もありません．
